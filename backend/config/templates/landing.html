Personal Brand Hub API Backend

About This API

This is the backend API for atonixdev's personal brand hub website. It provides RESTful endpoints for managing profile information, projects, services, testimonials, and contact forms.

Django 4.2 Django REST Framework PostgreSQL/SQLite JWT Authentication OpenStack AI/ML DevOps

API Endpoints

Authentication

POST /api/token/ - Obtain JWT token

POST /api/token/refresh/ - Refresh JWT token

Profile

GET /api/accounts/profile/ - Get profile information

PUT /api/accounts/profile/ - Update profile

Projects

GET /api/portfolio/projects/ - List all projects

GET /api/portfolio/projects/{id}/ - Get project details

Services

GET /api/services/ - List all services

GET /api/services/{id}/ - Get service details

Testimonials

GET /api/testimonials/ - List all testimonials

GET /api/testimonials/{id}/ - Get testimonial details

Contact

POST /api/contact/ - Send contact message

Blog

GET /api/blog/posts/ - List blog posts

GET /api/blog/posts/{id}/ - Get blog post details

POST /api/blog/posts/ - Create blog post

Getting Started

To interact with this API, you can use tools like:

  • cURL - Command line HTTP client
  • Postman - API testing tool
  • Frontend Application - React app running on port 3000

Example API Call:

curl -X GET "http://localhost:5000/api/accounts/profile/" \ -H "Authorization: Bearer YOUR_JWT_TOKEN" \ -H "Content-Type: application/json"
